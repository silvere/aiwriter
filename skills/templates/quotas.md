# 硬配额 + 段首路标词黑名单

> 当 aiwriter.md 主流程走到 Step 4 自检 ② 或 Step 6 第二刀（AI 味刀）时 Read 此文件。

## 硬配额表（跨全文累计，超标即修）

| 配额项 | 精炼版 | 标准版 | 深度版 | 超支处理 |
|------|------|------|------|---------|
| 加粗独立成段 | ≤ 2 | ≤ 3 | ≤ 4 | 保留最锋利的 |
| 一句话成段 | ≤ 2 | ≤ 4 | ≤ 6 | 合并回正常段落 |
| 段首路标词 | = 0 | = 0 | = 0 | 全部删除 |
| 金句加粗 | ≤ 2 | ≤ 3 | ≤ 4 | 同加粗独立成段 |

**金句配额与加粗配额对齐**：两者是同一件事的两个视角。

## 比喻质量测试（无数量上限，每个过两关）

- **陈腐测试**：这个比喻在过去一年内被 ≥3 篇爆款用过吗？
  - 已知陈腐清单（发现即禁用）：光纤泡沫 / 种树 / iPhone 时刻 / 元年 / 寒武纪 / 蒸汽机时刻 / 移动互联网前夜 / 工业革命级 / 基建期
  - 禁用后换具体事实直接说
- **替代测试**：把这个比喻删掉，用具体事实直接说，哪个更清楚？
  - 更清楚 → 比喻是多余的，删
  - 过两关 → 随便用

## 段首路标词黑名单（发现即删，用内容逻辑过渡替代）

- "注意" / "问题来了" / "翻译成人话" / "画风突变" / "你读到这里"
- "让我们来看" / "接下来" / "下面我们谈谈" / "首先 / 其次 / 最后"
- "有意思的是" / "值得一提" / "不难发现" / "不得不说" / "值得注意"
- "总的来说" / "综上所述" / "说到底"（出现在段末也要改）
- "写到这里" / "话又说回来" / "回到主线"
- "想必你已经发现" / "聪明的读者会问"

## AI 味 2.0 特征扫描（发现即改）

- **比喻通货膨胀**：同一段出现 ≥2 个比喻 → 删到 1 个或 0 个
- **段首指路成瘾**：任何段以路标词开头 → 删路标词，用内容逻辑过渡
- **加粗金句滥发**：相邻 3 段内 ≥2 处加粗 → 删到 1 处
- **"亲切口语"伪装**：识别"你读到这里应该意识到了"、"翻译成人话"、"画风突变"这类假亲切 → 删除或换说法
- **均质段落**：连续 3 段以上长度和语气相似 → 打散 1 段（短句/反问/具体场景）

**配额比"扫词"更重要**——即使每个词都不违规，如果全文 5 个比喻、6 处加粗，照样是 AI 味重灾区。
