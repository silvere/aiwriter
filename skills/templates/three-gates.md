# Step 2.5：三道门槛（详细规则）

> 当 aiwriter.md 主流程走到 Step 2.5 时 Read 此文件。
>
> 现在的 AI 写出肤浅文章的根本原因不是"问题问少了"，而是"没有门槛"——AI 会用深度词回答任何问题，答案看着深实际是套话。
> 本步骤用**三道硬门槛**代替走流程式自问自答。每道必过，否则角度重做。

## 门槛 A：竞品死刑

**目的**：杀死烂大街的角度。

1. WebSearch 同题 3-5 篇近期爆款（搜索词："[主题] 公众号"、"[主题] 深度分析"、"[主题] {当前年}"）
2. 对每篇用一句话写出其**核心判断**
3. 本文角度必须有**至少 1 个它们都没写过的判断**

**拒稿条件**：
- 本文核心判断已被 3 篇以上竞品覆盖 → **换角度**，不许"换措辞继续写"
- 核心类比（基建期/光纤泡沫/互联网早期/iPhone 时刻）在竞品里出现 ≥2 次 → **禁用该类比**

**输出格式**：
```
【竞品死刑】
- 竞品 1：[标题] → 核心判断："___"
- 竞品 2：[标题] → 核心判断："___"
- 竞品 3：[标题] → 核心判断："___"
- 本文增量判断："___"（解释为什么竞品都没写这个）
```

## 门槛 B：三位专家拒稿

**目的**：用真正有犀利判断框架的专家来压角度。

参考 `templates/expert-voices.md` 的挑选指南：

1. 根据主题挑 **3 位真实顶级专家**，视角互补（产业 + 技术/工程 + 社会/人文）
2. 每位模仿其真实语气给当前角度**最毒的一刀差评**
3. 对每刀给出**挡刀方案**——用论据、数据、案例、逻辑修正来挡，**禁用免责声明挡刀**（"这是本文讨论范围外"不算挡住）

**输出格式**：
```
【三位专家拒稿】

1. [专家 A 姓名]（[擅长视角]）
   > "[模仿其真实语气的一句差评]"
   刀口：[具体失败点]
   挡刀：[我们如何用论据/数据/案例修正]

2. [专家 B]（...）
3. [专家 C]（...）
```

**拒稿条件**：
- 三刀全挡住 → 过关
- 挡住两刀、一刀挡不住 → 调整角度覆盖那一刀
- 挡不住两刀及以上 → **回到 Step 2，重新找角度**

## 门槛 C：一刀测试

**目的**：全文只服务一个核心判断，确保这个判断是"一刀见血"而非"聪明综述"。

提炼本文**唯一的核心判断**（一句话，≤30 字），必须同时满足三条：

1. **可证伪**：如果 X 发生，我就错了。X 必须是具体可观测事件（不能是"如果形势逆转"这种虚词）
2. **反常识**：大多数人的直觉是反向的（至少 60% 的读者第一反应是"不对吧"）
3. **行动含义**：读完后读者能做一件**具体不同的事**（不能是"多思考"、"保持关注"这种废话）

**输出格式**：
```
【一刀测试】
核心判断："___"（≤30 字）
- 可证伪：如果 [具体事件 X] 发生，此判断错误
- 反常识：大多数人的直觉是 [反向的 Y]，本文说 [Z]
- 行动含义：读者应该从 [原行为] 切换到 [新行为]
```

**拒稿条件**：任一条不过 → 重写核心判断。不允许"先写起来再看"。

## 三道门槛全过 → 产出「角度声明板块」

```
【角度声明】

竞品死刑：
  - 3 篇竞品判断：[..., ..., ...]
  - 本文增量："___"

三位专家拒稿：
  - [专家 A] 刀口："___" → 挡刀："___"
  - [专家 B] 刀口："___" → 挡刀："___"
  - [专家 C] 刀口："___" → 挡刀："___"

一刀核心判断："___"
  - 可证伪条件：___
  - 反常识点：___
  - 行动含义：___

时效性：[上升期 / 峰值期 / 衰退期]，触发事件：___
审稿偏好：[自动四刀审稿并修复 / 审稿后我来决定 / 跳过审稿]
```

展示角度声明，**等待用户批准**。用户可要求换角度、换专家、重做任意一道门槛。
